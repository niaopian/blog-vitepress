# 常用git操作 <Badge type="warning" text="beta" />
mac环境下的git操作
## 安装
↓↓↓ **前置条件** ↓↓↓
- 已经安装brew
---


通过 brew 安装 git
```
brew install git
```

## 使用
基础设置
```
git config --global user.name   "你的名字或昵称"
git config --global user.email  "你的邮箱"
git config --list  "查看配置"
```
